# Recommended Extra Installs / Additions

Based on the current installed inventory, you already have a very strong setup.

## Worth considering

### 1. Official MCP server development plugin
If you plan to build your own MCP servers, consider Anthropic's official MCP server development plugin if it is available in your Claude Code marketplace.

Use case:
- build your own local connector
- expose custom engineering tools
- connect a personal database or internal tool

### 2. React Flow or equivalent graph library
Not a Claude skill, but recommended dependency for the app.
Use for:
- agent maps
- MCP connector maps
- workflow diagrams

### 3. Framer Motion
Recommended dependency for polished interactive UI.

### 4. shadcn/ui or a strong component library
Useful, but do not let it make the app look generic. Customize heavily.

### 5. A small icon library
Lucide is a good default for clean interface icons.

## What not to install now

Do not install more Claude skills just because they exist. Too many broad skills can make routing noisy. First finish the app with your existing design/frontend/planning/review stack.

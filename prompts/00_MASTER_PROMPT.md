# MASTER PROMPT FOR CLAUDE CODE

You are building a premium, visual, interactive educational web app called **Claude Code AI Visual Lab**.

## Mission

Create an app that explains the AI tooling ecosystem around Claude Code in a way that is visual, simple, interactive, and impressive enough to show as a portfolio-quality product.

The app should explain:

* Claude Code as an agentic coding tool
* agents and subagents
* skills
* MCP servers/connectors
* tokens and context windows
* how to reduce token waste
* when too many skills/agents become harmful
* how to structure a high-quality Claude Code workflow
* how to build beautiful apps using Claude Code

## Critical style requirement

This must NOT feel like documentation.

It should feel like an interactive AI control room / museum exhibit:

* minimal text
* strong visuals
* animated diagrams
* clickable modules
* interactive simulations
* visual metaphors
* polished UI
* clear hierarchy
* premium product feel

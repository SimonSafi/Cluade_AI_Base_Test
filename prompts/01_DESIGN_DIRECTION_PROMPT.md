# Design Direction Prompt

Use this prompt after the initial setup if the app looks too generic.

The current app is functionally on the right path, but the visual quality must become much more premium and alive.

Please run a design-focused pass using the relevant design skills.

Goal:
Create a custom visual identity called **AI Control Room / Interactive Museum**.

Improve:
- hero screen impact
- motion and micro-interactions
- card depth
- iconography
- diagrams
- contrast and hierarchy
- less text, more visual explanation
- better empty space
- more “wow” moment on first load

Do not just add gradients randomly.
Build a consistent design system:
- color tokens
- spacing tokens
- typography scale
- card variants
- diagram components
- motion rules
- reusable concept-node components

After the design pass:
1. show what changed
2. run the app
3. inspect visually
4. fix obvious visual defects
